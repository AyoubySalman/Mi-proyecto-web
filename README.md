1. Clona este repositorio:

    ```bash
    git clone https://github.com/usuario/nombre-del-proyecto.git
    ```

2. Entra en la carpeta del proyecto:
    ```bash
    cd nombre-del-proyecto
    ```

3. Instala las dependencias:
    ```bash
    npm install
    ```

4. Ejecuta la aplicación:
    ```bash
    npm start
    ```
